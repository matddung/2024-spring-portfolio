Project 1

MILY(May I Law You?)

팀 구성 : 4명
개발 기간 : 2023.10.10 - 2023.12.06
맡은 역할 : 유저·견적서·예약 아키텍처 설계 및 구현

각 분야에 맞는 글 작성 시 변호사가 답변할 수 있습니다.

각 분야에 맞는 항목을 선택하여 견적서를 변호사들에게 보낼 수 있습니다.

원하는 변호사와 상담을 예약할 수 있습니다.


개발 환경

언어 : Java (JDK 1.7)

프레임워크 : Spring Boot 3.1.4

DB : MariaDB

API, 라이브러리 : 토스 페이먼츠, SMTP
